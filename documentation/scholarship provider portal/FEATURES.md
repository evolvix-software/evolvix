## 1. Campaign Management Features

### 1.1 Create Campaign

#### Core Functionality
- **Campaign Form**: Multi-step wizard
- **Course Selection**: Link to existing courses or create pooled campaign
- **Eligibility Rules Builder**: Visual rule builder
- **Funding Configuration**: Set funding model and amounts
- **Slot Management**: Define total slots and reservation rules
- **Application Window**: Set open/close dates
- **Preview Mode**: Preview campaign before publishing

#### Advanced Features
- **Template Library**: Pre-built campaign templates
- **Duplicate Campaign**: Clone existing campaign
- **Bulk Campaign Creation**: Create multiple campaigns
- **Auto-close**: Automatic closure when slots filled
- **Campaign Analytics**: Track campaign performance

#### Eligibility Rules
- **Academic Criteria**: Min/max CGPA
- **Financial Need**: Income thresholds
- **Merit-Based**: Academic excellence
- **Need-Based**: Financial hardship
- **Course-Specific**: Link to specific courses
- **Geographic**: Location restrictions
- **Custom Rules**: Free-text criteria

---

### 1.2 Manage Campaigns

#### Campaign Listing
- **Grid/List View**: Toggle between views
- **Filters**: 
  - Status (Draft, Open, Closed, Completed)
  - Campaign Type
  - Date Range
  - Linked Courses
- **Search**: Full-text search
- **Sorting**: 
  - Date Created
  - Application Count
  - Funded Amount
  - Slots Available

#### Campaign Actions
- **Edit**: Modify campaign details
- **Duplicate**: Create copy
- **Close**: Close to new applications
- **Reopen**: Reopen closed campaign
- **View Applications**: Navigate to applications
- **View Scholars**: Navigate to awarded scholars
- **Analytics**: View campaign metrics

#### Campaign Status
- **Draft**: Not published
- **Open**: Accepting applications
- **Closed**: No longer accepting
- **Completed**: All slots awarded
- **Cancelled**: Campaign cancelled

## 2. Application Review & Verification

### 2.1 Application Queue

#### Queue View
- **Status Filters**: Submitted, Under Verification, Review Pending, Shortlisted, Awarded, Rejected
- **Priority Sorting**: High priority first
- **Bulk Selection**: Select multiple applications
- **Quick Actions**: Verify, Score, Award, Reject

#### Application Card
- **Student Info**: Name, email, CGPA
- **Campaign**: Campaign name
- **Status Badge**: Current status
- **Score**: Total score (if reviewed)
- **Flags**: Visual indicators for issues
- **Quick Actions**: View, Verify, Score, Award

---

### 2.2 Document Verification

#### Verification Panel
- **Document Viewer**: View uploaded documents
- **Verification Checklist**: 
  - Transcript verified
  - ID proof verified
  - Income proof verified
  - Background check completed
- **Verification Actions**:
  - Mark as Verified
  - Request More Info
  - Flag for Review
  - Reject Document
- **Verification Log**: Track all verification actions

#### Auto-Verification (AI)
- **Document Extraction**: Extract data from documents
- **CGPA Extraction**: Auto-extract CGPA from transcript
- **Income Extraction**: Auto-extract income from proof
- **Discrepancy Detection**: Flag inconsistencies
- **Verification Suggestions**: AI-powered suggestions

---

### 2.3 Application Scoring

#### Scorecard System
- **Scoring Rubric**:
  - Academic Score (0-100)
  - Financial Need Score (0-100)
  - Motivation Score (0-100)
- **Weighted Scoring**: Configurable weights
- **Total Score**: Calculated weighted score
- **Ranking**: Auto-rank by score
- **Reviewer Notes**: Add notes per section

#### Scoring Features
- **Multiple Reviewers**: Multiple reviewers can score
- **Average Score**: Calculate average if multiple reviewers
- **Score History**: Track score changes
- **Score Justification**: Required notes for scores

---

### 2.4 Award Decision

#### Award Process
- **Shortlist**: Move to shortlist
- **Award**: Award scholarship
- **Reject**: Reject application
- **Withdraw**: Withdraw application
- **Reopen**: Reopen closed application

#### Award Actions
- **Auto-Award**: Auto-award if score >= threshold
- **Manual Award**: Provider manually awards
- **Batch Award**: Award multiple at once
- **Award Letter**: Generate award letter
- **Notification**: Notify student of award
