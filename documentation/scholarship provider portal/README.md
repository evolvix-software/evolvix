## Scholarship Flow

```
1. Provider creates scholarship campaign
2. Students apply for scholarships
3. Provider reviews and verifies applications
4. Provider awards scholarships
5. Provider transfers funds to Evolvix
6. Evolvix disburses funds to courses
7. Provider tracks scholar progress (grades, assignments, engagement)
8. Scholar improves CGPA and completes course milestones
9. Scholar graduates with improved grades
10. Scholar gets placed in a job
11. Provider celebrates achievements (badges, certificates, success stories)
12. Provider shares success stories to encourage other scholars
13. Provider measures impact (ROI, graduation rates, job placement rates)
```
