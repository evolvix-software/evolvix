## Overview

The Career Page Builder allows employers to create and customize their public-facing career page to attract candidates and showcase company culture.

## Routes
```
/portal/employer/career-page          # Builder/Editor
/careers/[companySlug]                # Public career page
```

- Theme settings
- SEO settings

## Available Sections

### 1. Hero Section
**Purpose:** First impression, company branding

**Content:**
- Company logo
- Hero image/video background
- Headline text
- Subheadline text
- CTA button (View Jobs, Join Talent Pool)
- Optional: Video background

**Customization:**
- Background image/video
- Overlay opacity
- Text color
- CTA button style
- Height (Full screen, Custom)

### 2. About Section
**Purpose:** Company story and mission

**Content:**
- Section title
- Company description (rich text)
- Mission statement
- Vision statement
- Values list
- Company stats (optional)

**Customization:**
- Layout (Single column, Two column)
- Background color
- Text alignment
- Image placement

### 3. Values Section
**Purpose:** Showcase company culture

**Content:**
- Section title
- Values grid (3-6 values)
- Each value:
  - Icon/image
  - Title
  - Description

**Customization:**
- Number of columns
- Value card style
- Icon style
- Background

### 4. Benefits Section
**Purpose:** Highlight perks and benefits

**Content:**
- Section title
- Benefits list:
  - Icon
  - Benefit name
  - Description
- Benefits categories:
  - Health & Wellness
  - Professional Development
  - Work-Life Balance
  - Financial Benefits

**Customization:**
- Grid layout
- Card style
- Icon set
- Color scheme

### 5. Team Section
**Purpose:** Show team members and culture

**Content:**
- Section title
- Team member cards:
  - Photo
  - Name
  - Role
  - Bio/Quote
- Team photos gallery
- Culture photos

**Customization:**
- Layout (Grid, Carousel)
- Card style
- Photo size
- Show/hide details

### 6. Testimonials Section
**Purpose:** Employee testimonials

**Content:**
- Section title
- Testimonial cards:
  - Employee photo
  - Employee name
  - Employee role
  - Testimonial text
  - Rating (optional)

**Customization:**
- Layout (Carousel, Grid)
- Card style
- Auto-play carousel
- Show ratings

### 7. Jobs Section
**Purpose:** Display open positions

**Content:**
- Section title
- Job listings (auto-populated)
- Job filters (optional)
- "View All Jobs" link

**Customization:**
- Number of jobs shown
- Job card style
- Show filters
- Sort order

### 8. CTA Section
**Purpose:** Call-to-action for candidates

**Content:**
- Section title
- Description text
- CTA buttons:
  - View Open Jobs
  - Join Talent Pool
  - Contact Us
- Background image/color

**Customization:**
- Button styles
- Background
- Text alignment
- Layout

### 9. Contact Section (Optional)
**Purpose:** Contact information

**Content:**
- Section title
- Office address
- Contact email
- Phone number
- Social media links
- Contact form (optional)
